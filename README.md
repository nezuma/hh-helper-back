# NEW PROJECT

## SET PREFERENCES FOR WORKSPACE

1. Install Prettier - Code formatter https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode
2. Use shortcut `ctrl+shift+p` and insert `user settings`
3. Choose `Preferences: Open User Settings (JSON)` and add lines:

```json
"editor.defaultFormatter": "esbenp.prettier-vscode",
"editor.formatOnSave": true,
"notebook.formatOnSave.enabled": true
```

4. Install TypeScript Barrel Generator https://marketplace.visualstudio.com/items?itemName=eliostruyf.vscode-typescript-exportallmodules
